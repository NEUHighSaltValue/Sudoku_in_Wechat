# 微信数独小程序 （数独大乱斗）

## 目前该项目已经获得了微信小程序高校大赛，全国二等奖（第六名）的优异成绩。

最新的版本支持了简单的解数独、高亮提醒相同数字、错误提示、选择做题顺序、多人pk、战绩查询等众多功能，此外在细节上，我们也做了页面onHide 时自动暂停，onShow 时自动继续等贴心功能。

---

## 这是一个基于微笑小程序开发的数独游戏，一方面是学习最近十分火热的微信小程序开发，另一方面也作为软件工程的项目作业来提交。
